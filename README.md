## ğŸš€ Getting Started

### Prerequisites
- VS Code with Live Preview extension (already installed)
- Modern web browser (Chrome, Firefox, Safari, Edge)

### Running the Website

1. **Using VS Code Live Preview** (Recommended):
   - Open the project in VS Code
   - Right-click on `index.html`
   - Select "Show Preview" or use Ctrl+Shift+V
   - The website will open in VS Code's built-in browser

2. **Using Live Server Extension**:
   ```vscode-extensions
   ritwickdey.liveserver
   ```
   - Install the Live Server extension if preferred
   - Right-click on `index.html`
   - Select "Open with Live Server"

3. **Direct File Opening**:
   - Simply double-click `index.html` to open in your default browser
   - Note: Some features may not work due to file:// protocol restrictions

## ğŸ¨ Design System

### Color Palette
- **Primary Blue**: #2c5aa0 (Navigation, buttons, accents)
- **Dark Blue**: #1e3f73 (Hover states, headers)
- **Dark Gray**: #2c3e50 (Text headings)
- **Medium Gray**: #555 (Body text)
- **Light Gray**: #f8fafc (Background sections)
- **Success Green**: #27ae60 (Success states, checkmarks)

### Typography
- **Font Family**: Inter (Google Fonts)
- **Headings**: 600 weight, various sizes (3rem down to 1.5rem)
- **Body Text**: 400 weight, 16px base size
- **Line Height**: 1.6 for readability

### Components
- **Buttons**: Rounded corners (5px), hover animations
- **Cards**: Subtle shadows, 10-15px border radius
- **Forms**: Clean inputs with focus states
- **Navigation**: Fixed header with scroll effects
